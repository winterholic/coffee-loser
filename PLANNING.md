# 너 오늘 커피사라 (Coffee-Loser)

## 프로젝트 개요

다양한 미니게임을 통해 벌칙자, 당첨자, 팀 등을 재미있게 정할 수 있는 웹앱

## 기술 스택

- **프레임워크**: React.js
- **빌드 도구**: Vite
- **언어**: TypeScript
- **파비콘**: coffee-loser.ico
- **로고**: coffee-loser.svg

# 게임 상세 기획

---

## 탭 1: 끝까지 가야대 (경주 게임)

마블룰렛 스타일의 달리기 경주 게임

### 입력 방식
- 마블룰렛처럼 `(짱구*5, 짱아*10, 봉미선*3)` 형식으로 항목과 개수 입력
- 당첨 기준 선택: 1등(처음 도착) 또는 꼴등(마지막 도착)

### 게임 진행
- 각 항목별로 다양한 색깔의 말(horse)이 생성됨
- 말 위에 항목 이름 표시
- 평면 트랙에서 달리기 경주 진행
- 각 말의 속도가 시시각각 변화
- 랜덤 장애물이 중간중간 등장하여 진행 방해

### 말 배치 규칙
- **한 라인에 여러 말 배치 가능**: 말 개수가 많으면 같은 라인에 여러 말이 동시에 달림
- **말 간 통과 허용**: 말들은 서로 겹쳐서 지나갈 수 있음 (충돌 없음)
- 라인 수는 화면 크기와 말 개수에 따라 자동 조절

### 화면 구성
```
┌──────────────────────────────────────────────────────────────────────┐
│  START                                                    FINISH     │
│  ════════════════════════════════════════════════════════════════    │
│     🐴짱구  🐴짱구     ──🪨──    🐴짱구                              │
│  ════════════════════════════════════════════════════════════════    │
│  🐴짱아   🐴짱아  🐴짱아   🐴짱아  ──🪨──   🐴짱아  🐴짱아           │
│  ════════════════════════════════════════════════════════════════    │
│          🐴봉미선              🐴봉미선        ──🪨──                │
│  ════════════════════════════════════════════════════════════════    │
│                                                                      │
│    [속도 변화 중...]                              [장애물 등장!]      │
└──────────────────────────────────────────────────────────────────────┘

※ 같은 이름의 말이 여러 개 존재 (짱구*5 → 짱구 말 5개)
※ 말들은 서로 통과 가능 (겹침 허용)
```

### 결과 화면
- 1등 또는 꼴등 말의 이름 강조 표시
- 전체 순위 목록 제공

## 탭 3: 커피 누가사? (벌칙자 정하기)

### 입력 단계
1. 총 인원수 입력
2. 각 사람 이름 입력
3. 벌칙 받을 인원수 입력
4. 벌칙 종류 개수 입력
5. 벌칙 종류 선택/입력

### 벌칙 종류 (프리셋)
| 카테고리 | 벌칙 |
|---------|------|
| 사기 | 커피사기, 밥사기, 술사기, 숙취해소제 사기, 코인노래방 사기, 아이스크림 사오기 |
| 청소 | 설거지하기, 쓰레기 버리기, 화장실 청소하기, 퇴실 청소하기 |
| 퍼포먼스 | 춤 30초 추기, 애교 10초 하기, 건배사 하기 |
| 기타 | 직접 입력 |

### 결과 화면
```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│                      🎉 결과 발표! 🎉                           │
│                                                                 │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐           │
│  │ ░░░░░░░ │  │ ▓▓▓▓▓▓▓ │  │ ░░░░░░░ │  │ ▓▓▓▓▓▓▓ │           │
│  │ ░ 하얀 ░ │  │ ▓ 빨간 ▓ │  │ ░ 하얀 ░ │  │ ▓ 빨간 ▓ │           │
│  │ ░ 쪽지 ░ │  │ ▓ 쪽지 ▓ │  │ ░ 쪽지 ░ │  │ ▓ 쪽지 ▓ │           │
│  │ ░░░░░░░ │  │ ▓▓▓▓▓▓▓ │  │ ░░░░░░░ │  │ ▓▓▓▓▓▓▓ │           │
│  │         │  │         │  │         │  │         │           │
│  │  철수   │  │  영희   │  │  민수   │  │  지영   │           │
│  │         │  │ ──────  │  │         │  │ ──────  │           │
│  │  안전!  │  │커피사기 │  │  안전!  │  │ 밥사기  │           │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘           │
│                                                                 │
│                      [다시하기] [공유하기]                       │
└─────────────────────────────────────────────────────────────────┘
```

- **하얀 쪽지 (░)**: 벌칙 미당첨자 (이름만 표시)
- **빨간 쪽지 (▓)**: 벌칙 당첨자 (이름 + 벌칙 내용 표시)

## 탭 4: 팀나누기

### 입력

### 옵션

## 광고 배치 전략

### 자연스러운 광고 위치
1. **메인 페이지**: 게임 목록 중간 (3번째와 4번째 게임 사이)
2. **게임 페이지**: 게임 결과 하단
3. **전환 시점**: 게임 완료 후 결과 확인 전 (인터스티셜, 선택적)

### 광고 사이즈
- 모바일: 320x100 (반응형)
- 데스크톱: 728x90 (리더보드)
- 인터스티셜: 전체화면 (선택적)

### 광고 원칙
- 게임 플레이를 방해하지 않는 위치
- 콘텐츠와 자연스럽게 어우러지는 디자인
- 과도한 광고로 사용자 경험 저해 금지

## 공통 UI/UX 고려사항

### 반응형 디자인
- 모바일 우선 설계
- 데스크톱 확장 지원
- 탭 네비게이션은 모바일에서 스와이프 가능

- 게임 진행 시 부드러운 애니메이션
- 결과 발표 시 효과 (confetti, 강조 효과 등)
- 탭 전환 시 슬라이드 효과
